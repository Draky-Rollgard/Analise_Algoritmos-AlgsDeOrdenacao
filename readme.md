# Análise de Algoritmos
`Analise_Algoritmos-AlgsDeOrdenacao` 

Repositório criado com intuito de guardar atividades desenvolvidas aplicáveis à Engenharia de Computação/Ciências da Computação que avaliam a complexidade temporal e espacial de algoritmos. Somente serão utilizados exemplos simples de algoritmos de ordenação já existentes.


## Estrutura de diretórios

Aqui você encontrará a seguinte estrutura de diretórios:

>Analise_Algoritmos-AlgsDeOrdenacao
>
>readme.md: arquivo atual
>>Arq_nome_alg: Diretório com nome do respectivo algoritmo de ordenação
>>>code: códigos de exemplo, desenvolvidos em `C`
>>
>>>docs: arquivos a serem apresentados
>>
>>>readme.md: resumo para leitura
